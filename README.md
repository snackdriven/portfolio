## What's here

Two-page portfolio site. Vanilla HTML and CSS, no frameworks, no build step. It didn't need React and I refused to give it React.

- `index.html` — resume (experience, stack, what I'm looking for)
- `projects.html` — project writeups with tech details
- `resume/` — two variants (automation, tooling) in PDF and DOCX

Print styles built in. The resume prints to a clean single-page PDF with a light theme and ATS-friendly fonts.

## About me

I write tools to kill manual work. Came up through QA: eight behavioral health products, no backup. Every slow process was mine to eliminate or live with. 14 scripts later. Most of my repos are private because they're either half-finished or contain my unhinged commit messages.
