02
Jira Local
2025
50 tickets loaded in under 2 seconds, 5 view modes, keyboard-first
Holiday break, backlog of piled-up tickets, most of them queued for release within a day or two of getting back. JIRA's default view wasn't cutting it. The ticket names didn't have the context I needed to sort them at a glance, and I couldn't reorder things without changing the actual board, which would mess with everyone else's setup. Built a local wrapper instead: 5 views (board, list, tree, detail, calendar), custom sorting, full keyboard navigation. React Query caching loads 50 tickets in under 2 seconds.

03
chronicle
2025
Timeline memory system with direct MCP integration for Claude
I kept losing context between sessions. Not just what I decided, but why, and what was going on at the time. It came up most when I'd hit a known bug I'd already called out in a previous sprint, or try to remember a decision made in a ticket that nobody documented. Chronicle tracks all of that. The main draw is MCP integration: Claude can query my personal history directly mid-conversation, which is most of why I built it. Key-value storage for quick lookups, full-text search and tag filtering when I need to dig. TanStack Table for the timeline view, Express for the API.

05
OpenClaw
2025–present
5 AI agents, one HTTP gateway, 87% cost reduction ($120/month → $15)
Coordinates 5 AI agents with isolated workspaces. The gateway routes messages to the right specialist: task management, scheduling, and a narration agent that keeps things moving without being naggy. Each agent only sees what it needs.
Model routing depends on what's being asked. Cheap models for short requests, heavier models when something needs more reasoning. Before this, everything was hitting the Claude API directly and the bill showed it. Monthly cost is now $5-15. The HTTP gateway handles auth and session state, plus cross-agent relay when needed.
