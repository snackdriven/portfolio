<p class="lead">I had a research agent read my old repos and put them in order. I already knew the order. I wanted to see if it could.</p>

It couldn&rsquo;t, and it didn&rsquo;t say so. It handed back a clean, confident timeline and decided inside-weather was the original, the root the rest grew out of, from around 2022. inside-weather is from this April. One of the newest things I own. I built it. I remember building it.

The agent had no date, so it picked one, set a small question mark next to the year like a tasteful hedge, and wrote a whole origin story on top of the number it had just invented. The hedge was the tell. A real memory doesn&rsquo;t come with a question mark. It comes with the wrong details and total confidence about them. This came with perfect confidence and one apologetic mark of punctuation.

The unsettling part wasn&rsquo;t that it was wrong. I knew it was wrong. It&rsquo;s that the wrong version was clean. Tidier than the truth. And it was dating work another machine had typed for me in the first place: one model inventing the birthday of what a different model built. Neither of them was in the room when it happened. I was.

We both got asked the same thing, when was inside-weather built. I knew. The machine didn&rsquo;t, and it was the machine that answered like it was sure. The git timestamp didn&rsquo;t answer like anything. It was just right.
